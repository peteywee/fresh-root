2. **Product Owner**: Success criteria + constraints
3. **Systems Architect**: Design + interfaces
4. **Security Red Team**: Threat model + veto
5. **Research Analyst**: Verify + tool deployment

- **E**: Security Veto + Reflexion (VERIFY)
