# Project Overview

Fresh Root is a staff scheduling platform built to be reliable, secure, and maintainable. This page
offers short, audience-specific summaries without exposing internal surface details.

## Choose your level

### 1) Non-technical (30 seconds)

Fresh Root helps teams create and manage schedules with a focus on reliability and operational
clarity. It is designed for teams that need predictable workflows and strong safeguards.

### 2) Product + operations (2 minutes)

Fresh Root emphasizes consistent scheduling workflows, auditability, and operational readiness. It
is built for teams that need reliable scheduling and clear release practices.

### 3) Engineering (5 minutes)

Fresh Root is a modern web application with a secured API layer, shared libraries for validation and
UI, and a structured documentation system. The project favors validated inputs, consistent
interfaces, and clear operational guidance for deployments and testing.
