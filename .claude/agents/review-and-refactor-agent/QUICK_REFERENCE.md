# Review & Refactor Agent — Quick Reference

## Invocation
```
Use the review and refactor agent to refactor [target]
```

## Process
1. Load guidelines
2. Review code
3. Refactor to match standards
4. Run tests to verify

## Constraints
- ✅ Keep files intact
- ✅ Ensure tests pass
- ✅ Clean and maintainable
- ✅ Follow guidelines

## Output
- Refactored code
- All tests passing
- No regressions
